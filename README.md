# shoehorn-ingest-test1

A simple test repository for data ingestion testing.

## Running Tests

To run the test suite:

```bash
python test_runner.py
```

## Test Structure

The repository includes:
- `test_runner.py` - Main test runner with basic test cases
- `requirements-test.txt` - Test dependencies (currently minimal)

## What Tests Cover

- Basic functionality validation
- String processing operations  
- List operations
- Test framework functionality
